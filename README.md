# Lieferspatz
Lieferspatz is a web application developed as part of a university project.  
The idea is similar to platforms like *Lieferando*, focusing on online food ordering and delivery workflows.

## Tech Stack
- Python (Flask)
- MySQL
- HTML, CSS
- JavaScript
- PHP (for frontend integration)

## Features
- Backend logic implemented with Flask (`app.py`)
- Database design and SQL queries using MySQL
- Dynamic web pages with HTML, CSS, and JavaScript
- Order and data handling via a relational database
- MVC-like project structure (models, templates, static files)

## Project Structure
```text
Lieferspatz/
├── app.py
├── models.py
├── create_tables.py
├── templates/
├── static/
└── README.md
```
## How to Run Locally
1. Clone the repository  
2. Create and activate a virtual environment  
3. Install required dependencies  
4. Configure the database connection  
5. Run the application:
```bash
python app.py
```
